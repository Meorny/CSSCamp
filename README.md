## üîß Using CSSCamp in Your Project

If you want to create your own web page using CSSCamp, follow these steps:

### 1. Create a New HTML File

- Open your text editor.
- Create a new file and save it as `my-page.html`.

### 2. Include CSSCamp Styles

Copy this line into the `<head>` section of your HTML file to link the CSSCamp framework:

```html
<link rel="stylesheet" href="path/to/csscamp.css">
```

Replace `"path/to/csscamp.css"` with the actual path to the CSS file from the extracted folder.

### 3. Add Your Content

In the `<body>` of your file, write your page content using normal HTML tags. Use classes from CSSCamp to style elements like buttons, forms, or headings. For example:

```html
<button class="cc-btn-primary">Click me</button>
```

### 4. Open Your Page in a Browser

- Save your file.
- Double-click it to open in your browser and see the styles applied.
